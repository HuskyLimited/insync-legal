Insync Post

A personal desktop scheduler for posting short-form video to a single TikTok creator account at researched optimal times. Drop clips into a queue, pick a window, fill in the engagement levers, let it post on its own.

What it does

  • Imports your video clips with auto-probe (duration, dimensions, codec) and thumbnail extraction.
  • Computes optimal posting windows from your own historical engagement data using TikTok's Display API.
  • Composes posts with caption, hashtags, pinned comment, cover frame, hook-frame inspector, length advisor, and chess-format quick-templates — every lever is research-backed.
  • Schedules to the calendar via drag-and-drop. The local APScheduler fires at slot time and uploads via TikTok's Content Posting API.

How it uses TikTok's APIs

Login Kit for one-time OAuth so the operator can authenticate their own TikTok account. Display API to read the operator's own past post stats for time-of-day analytics. Content Posting API to upload the operator's own pre-recorded clips at scheduled times.

The application stores OAuth tokens locally on the user's machine. It does not share data with any third party and operates only on behalf of the authenticated user against their own account.

Status

Local-only desktop application built with Tauri + Vue + FastAPI + SQLite. Single user. Personal scheduling tool, not a multi-tenant platform.
